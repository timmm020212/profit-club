# 🚀 **Простые команды для запуска**

## ✅ **Без слэшей и точек - максимально просто!**

## 🎯 **Простые команды:**

### **1. Запуск базы данных**
```
net start postgresql-x64-15
```

### **2. Запуск сайта**
```
dev
```

### **3. Запуск ботов**
```
masters-bot
client-bot
```

### **4. Установка зависимостей**
```
npm-simple install
```

## 🔄 **Полный порядок запуска:**

```
# Терминал 1: База данных
net start postgresql-x64-15

# Терминал 2: Сайт
dev

# Терминал 3: Мастерский бот
masters-bot

# Терминал 4: Клиентский бот
client-bot
```

## 🌐 **Адреса после запуска:**
- **Сайт**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **Мастерский бот**: @ProfitClub_staff_bot
- **Клиентский бот**: @ProfitClub_club_bot

## 📁 **Созданные файлы:**
- `dev.bat` - запуск сайта
- `masters-bot.bat` - запуск мастерского бота
- `client-bot.bat` - запуск клиентского бота
- `npm-simple.bat` - для npm команд

## 🎯 **Самый простой запуск:**

1. Откройте терминал
2. Введите `dev` - сайт запустится
3. Откройте новый терминал
4. Введите `masters-bot` - бот запустится
5. Откройте новый терминал
6. Введите `client-bot` - второй бот запустится

**Больше никаких слэшей и точек! Просто `dev` и `masters-bot`!** 🎉
