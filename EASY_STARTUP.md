# 🚀 **Простой запуск проекта**

## 🎯 **Теперь можно использовать обычные команды!**

### **📁 Созданы bat файлы:**
- `npm.bat` - для npm команд
- `start-dev.bat` - для запуска сайта
- `start-masters-bot.bat` - для мастерского бота
- `start-client-bot.bat` - для клиентского бота

## 🔄 **Команды для запуска:**

### **1. Запуск базы данных**
```bash
net start postgresql-x64-15
```

### **2. Запуск сайта**
```bash
# Теперь можно использовать обычную команду!
npm run dev

# Или просто запустить bat файл:
start-dev.bat
```

### **3. Запуск ботов**
```bash
# Мастерский бот
npm run masters-bot
# или
start-masters-bot.bat

# Клиентский бот  
npm run client-bot
# или
start-client-bot.bat
```

### **4. Другие npm команды**
```bash
npm install
npm run build
npm test
```

## 🎯 **Быстрый старт:**

```bash
# 1. База данных
net start postgresql-x64-15

# 2. Сайт (в новом терминале)
npm run dev

# 3. Мастерский бот (в новом терминале)
npm run masters-bot

# 4. Клиентский бот (в новом терминале)
npm run client-bot
```

## 🌐 **Адреса после запуска:**
- **Сайт**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **Мастерский бот**: @ProfitClub_staff_bot
- **Клиентский бот**: @ProfitClub_club_bot
