# 🚀 **Быстрые команды запуска**

## ⚡ **Два способа запуска:**

### **Способ 1: С точкой (рекомендуется)**
```
.\dev
.\masters-bot
.\client-bot
```

### **Способ 2: Добавить в PATH (постоянно)**
```
# Запустите один раз:
setup-commands

# Перезапустите PowerShell и используйте:
dev
masters-bot
client-bot
```

## 🎯 **Быстрый старт:**

```
# 1. База данных
net start postgresql-x64-15

# 2. Сайт
.\dev

# 3. Мастерский бот
.\masters-bot

# 4. Клиентский бот
.\client-bot
```

## 🌐 **Адреса:**
- **Сайт**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **Мастерский бот**: @ProfitClub_staff_bot
- **Клиентский бот**: @ProfitClub_club_bot

## 💡 **Почему нужна точка?**
PowerShell требует `.\` для запуска файлов из текущей директории для безопасности.

**Выберите способ 1 для быстрого старта или способ 2 для постоянного удобства!** 🎉
