## Папки

### `app/` — Next.js App Router (страницы и API)

```
app/
├── page.tsx                 ← Главная страница (список услуг + запись)
├── layout.tsx               ← Корневой layout (шрифты, провайдеры)
├── globals.css              ← Глобальные стили
├── admin/
│   └── page.tsx             ← Панель администратора (тайм-лайн, услуги, мастера)
├── booking/
│   └── [serviceId]/page.tsx ← Страница записи на конкретную услугу
├── telegram-webapp/
│   └── edit/page.tsx        ← Редактирование профиля внутри Telegram WebApp
└── api/
    ├── services/            ← GET список, POST создать, PUT/DELETE /[id], PUT /order
    ├── appointments/        ← GET список, POST создать, PATCH /[id] статус
    ├── available-slots/     ← GET свободные слоты (?debug=1 для диагностики)
    ├── masters/             ← CRUD мастеров
    ├── work-slots/          ← CRUD рабочих слотов
    ├── clients/
    │   ├── register/        ← Регистрация клиента (→ pendingClients)
    │   ├── login/           ← Вход по phone+password
    │   ├── verify-telegram-code/ ← Привязка Telegram
    │   └── verify-status/   ← Проверка привязки
    ├── upload/              ← Загрузка фото услуг → public/uploads/services/
    ├── auth/                ← NextAuth (admin login)
    └── reminders/           ← Отправка напоминаний
```

### `components/` — React компоненты

```
components/
├── BookingServicesGrid.tsx  ← Список услуг + фильтры-чипы + FLIP анимация
├── BookingServiceCard.tsx   ← Карточка одной услуги (фото, цена, badge)
├── BookingModal.tsx         ← Форма записи (выбор мастера, даты, времени)
├── Header.tsx               ← Шапка сайта (поиск, кнопки входа)
├── RegistrationModal.tsx    ← Модалка регистрации/входа клиента
├── RegistrationProvider.tsx ← Context + авто-открытие при ?tg_code
├── AdminSiteServicesManager.tsx ← CRUD услуг в /admin
├── AdminDateSelector.tsx    ← Карусель дат в /admin (window ≠ selected)
├── AdminWorkSlotManager.tsx ← Управление рабочими слотами
├── AdminWorkSlotChangeRequests.tsx ← Запросы на изменение от мастеров
├── AdminTimeline.tsx        ← Тайм-лайн расписания
├── AdminSelect.tsx          ← Кастомный select в стиле админки
└── ui/                     ← shadcn/ui компоненты
```

### `db/` — База данных

```
db/
├── index.ts                 ← Реэкспорт из index-sqlite.ts (менять на postgres при миграции)
├── index-sqlite.ts          ← АКТИВНЫЙ: подключение к SQLite через better-sqlite3
├── index-postgres.ts        ← ГОТОВ: подключение к PostgreSQL (для миграции)
├── index-mock.ts            ← Мок для тестов (не использовать в prod)
├── schema-sqlite.ts         ← АКТИВНАЯ схема: все таблицы SQLite
└── schema.ts                ← Реэкспорт schema-sqlite.ts
```

**Импорт в коде:** `import { db } from "@/db"` или `import { db } from "../db/index-sqlite"` (в ботах)

### `telegram-bot/` — Telegram боты

```
telegram-bot/
├── admin-bot-full.ts        ← Бот администратора (уведомления о записях)
├── masters-bot-full.ts      ← Бот мастеров (confirm/reject слотов, запросы изменений)
├── client-simple.ts         ← Бот клиентов (регистрация, мои записи, о нас)
└── index.ts                 ← (не используется в production)
```

**Запуск:** через отдельные npm скрипты (не вместе с dev-сервером)

## Переменные окружения

| Переменная | Описание | Пример |
|-----------|----------|--------|
| TELEGRAM_BOT_TOKEN | Клиентский бот | 123456:ABC... |
| MASTERS_BOT_TOKEN | Бот мастеров | 789012:XYZ... |
| NEXTAUTH_URL | URL сайта | http://localhost:3000 |
| NEXTAUTH_SECRET | Секрет NextAuth | случайная строка 32+ символа |
| ADMIN_USERNAME | Логин администратора | admin |
| ADMIN_PASSWORD | Пароль администратора | securepassword |

## npm команды

```bash
npm run dev              # Запуск Next.js сервера (0.0.0.0:3000)
npm run bot:client       # Клиентский Telegram-бот
npm run bot:masters      # Бот мастеров
npm run bot:start        # Бот администратора
npm run db:push          # Применить схему к БД
npm run db:studio        # Drizzle Studio (просмотр БД)
npm run db:seed          # Заполнить услуги
npm run reminders:send   # Отправить напоминания вручную
npm run build            # Сборка для продакшна
```
