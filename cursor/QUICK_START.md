# 🚀 Быстрый старт для тестирования

## ✅ Все уже настроено!

База данных настроена, мастера добавлены, бот запущен. Можно тестировать!

## 📱 Тест 1: Telegram-бот

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Если видите меню - всё работает! ✅

**Добавьте рабочий день:**
- Нажмите "➕ Добавить рабочее время"
- Выберите мастера
- Дата: `25.12.2024`
- Время начала: `08:00`
- Время окончания: `18:00`

## 🌐 Тест 2: API (через браузер или скрипт)

### Вариант А: Быстрая проверка в браузере

Откройте эти ссылки (если Next.js запущен):

1. Мастера: http://localhost:3000/api/masters
2. Услуги: http://localhost:3000/api/services
3. Рабочие дни: http://localhost:3000/api/work-slots

### Вариант Б: Автоматический тест

Запустите скрипт:
```bash
npx tsx scripts/test-api.ts
```

## 📋 Минимальный тест-сценарий

1. **Добавьте рабочий день через бота:**
   - Мастер: любой
   - Дата: 25.12.2024
   - Время: 08:00 - 18:00

2. **Проверьте доступные слоты:**
   ```
   http://localhost:3000/api/available-slots?masterId=10&serviceId=1&date=2024-12-25
   ```
   (Замените masterId и serviceId на реальные из БД)

3. **Создайте запись через API** (используйте curl или Postman)

4. **Проверьте, что слот исчез** - запросите доступные слоты снова
