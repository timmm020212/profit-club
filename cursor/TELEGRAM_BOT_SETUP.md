# Инструкция по настройке и запуску Telegram-бота для админа

## Что было создано

1. **Схема базы данных:**
   - Таблица `masters` - мастера салона
   - Таблица `work_slots` - рабочие слоты времени для мастеров
   - Таблица `admins` - администраторы Telegram

2. **Telegram-бот** (`telegram-bot/index.ts`):
   - Проверка прав администратора
   - Просмотр списка мастеров
   - Добавление рабочего времени для мастеров
   - Просмотр расписания

3. **API endpoints для сайта:**
   - `/api/masters` - список всех активных мастеров
   - `/api/work-slots?masterId=1&date=2024-12-25` - доступные слоты времени

## Шаги по настройке

### 1. Применить схему базы данных

Выполните команду для создания таблиц в БД:

```bash
npm run db:push
```

Эта команда создаст таблицы:
- `masters`
- `work_slots`
- `admins`

### 2. Добавить мастеров в базу данных

Выполните команду для заполнения БД мастерами:

```bash
npm run db:seed-masters
```

Эта команда добавит всех 15 мастеров и администратора с ID `6140319824`.

### 3. Настроить переменные окружения (опционально)

Если хотите использовать переменные окружения вместо значений по умолчанию, создайте файл `.env.local` и добавьте:

```env
TELEGRAM_BOT_TOKEN=8568554790:AAEHlp0un2EoHLGSJlE2G-suTZKp5seXz30
ADMIN_TELEGRAM_ID=6140319824
DATABASE_URL=postgresql://postgres:postgres123@localhost:5433/profit_club
```

**Токен бота и ID администратора уже прописаны в коде по умолчанию**, так что этот шаг можно пропустить.

### 4. Запустить Telegram-бота

В отдельном терминале выполните:

```bash
npm run bot:start
```

Бот запустится и будет готов к работе. В консоли вы увидите сообщение "Bot started successfully!".

### 5. Начать работу с ботом

1. Откройте Telegram
2. Найдите вашего бота (имя бота будет то, которое вы указали при создании через @BotFather)
3. Отправьте команду `/start`
4. Если ваш Telegram ID соответствует администратору (`6140319824`), вы увидите главное меню

## Функционал бота

### Главное меню

- **📋 Список мастеров** - показать всех мастеров салона
- **➕ Добавить рабочее время** - добавить доступные слоты времени для мастера
- **📅 Просмотр расписания** - показать все добавленные рабочие слоты

### Добавление рабочего времени

1. Нажмите "➕ Добавить рабочее время"
2. Выберите мастера из списка
3. Введите дату в формате ДД.ММ.ГГГГ (например, 25.12.2024)
4. Введите время начала в формате ЧЧ:ММ (например, 10:00)
5. Введите время окончания в формате ЧЧ:ММ (например, 11:00)
6. Бот сохранит слот и покажет подтверждение

### Просмотр расписания

Нажмите "📅 Просмотр расписания" - бот покажет все рабочие слоты, сгруппированные по мастерам и датам.

## Синхронизация с сайтом

После добавления рабочего времени через бота, эти данные автоматически доступны на сайте через API:

- `GET /api/masters` - получить список мастеров
- `GET /api/work-slots` - получить все доступные слоты
- `GET /api/work-slots?masterId=1` - слоты конкретного мастера
- `GET /api/work-slots?date=2024-12-25` - слоты на конкретную дату
- `GET /api/work-slots?masterId=1&date=2024-12-25` - слоты мастера на дату

## Структура данных

### Мастер (Master)
```typescript
{
  id: number;
  fullName: string;           // Полное имя мастера
  specialization: string;     // Специализация
  telegramId: string | null;  // Telegram ID (если привязан)
  isActive: boolean;          // Активен ли мастер
  createdAt: Date;
}
```

### Рабочий слот (WorkSlot)
```typescript
{
  id: number;
  masterId: number;           // ID мастера
  workDate: string;           // Дата в формате YYYY-MM-DD
  startTime: string;          // Время начала в формате HH:MM
  endTime: string;            // Время окончания в формате HH:MM
  isAvailable: boolean;       // Доступен ли для записи
  createdAt: Date;
  createdBy: string;          // Telegram ID администратора
}
```

## Важные замечания

1. **Бот работает только для администраторов** - проверяется Telegram ID пользователя
2. **Добавленные слоты сразу доступны на сайте** - синхронизация происходит через БД
3. **Только будущие даты** - API показывает только слоты с датой >= сегодня
4. **Время окончания должно быть больше времени начала** - бот проверяет это автоматически

## Устранение проблем

### Бот не отвечает

1. Проверьте, что бот запущен (`npm run bot:start`)
2. Убедитесь, что токен бота правильный
3. Проверьте подключение к базе данных

### "У вас нет доступа"

Убедитесь, что ваш Telegram ID добавлен в таблицу `admins`. По умолчанию добавлен ID `6140319824`.

Чтобы узнать свой Telegram ID:
1. Найдите бота @userinfobot в Telegram
2. Отправьте ему `/start`
3. Он покажет ваш ID

Затем добавьте себя в БД через SQL или создайте отдельный скрипт.

### Ошибки подключения к БД

1. Убедитесь, что PostgreSQL запущен
2. Проверьте `DATABASE_URL` в `.env.local`
3. Проверьте, что база данных `profit_club` создана
