# Как запустить Telegram бота

## Проблема
Вы видите логи Next.js сервера (веб-сайта), но бот - это отдельная программа, которая должна работать в **отдельном терминале**.

## Решение

### Шаг 1: Откройте НОВЫЙ терминал
- В VS Code/Cursor: нажмите `Ctrl + Shift + `` (обратная кавычка) или `Ctrl + J`
- Или откройте новый PowerShell/Command Prompt

### Шаг 2: Перейдите в папку проекта
```powershell
cd "C:\Users\Fortn\Downloads\Profit Club"
```

### Шаг 3: Запустите бота
```powershell
npm run bot:start
```

### Шаг 4: Оставьте терминал открытым
В этом терминале вы увидите логи бота:
- `Starting Telegram bot...`
- `✅ Bot started successfully!`
- `🚀 /start HANDLER CALLED!` (когда кто-то отправит /start)
- `Generated verification code: 123456`

## Итого у вас должно быть 2 терминала:

1. **Терминал 1**: Next.js сервер (`npm run dev`) - логи веб-сайта
2. **Терминал 2**: Telegram бот (`npm run bot:start`) - логи бота

## Проверка работы бота

После запуска бота в отдельном терминале:
1. Отправьте `/start` в Telegram боту
2. В терминале с ботом должно появиться: `🚀 /start HANDLER CALLED!`
3. Бот должен отправить вам 6-значный код

## Если бот не отвечает

1. Проверьте, что в логах бота есть `Bot started successfully!`
2. Если нет - проверьте ошибки в логах
3. Убедитесь, что PostgreSQL запущен (Docker контейнер)
