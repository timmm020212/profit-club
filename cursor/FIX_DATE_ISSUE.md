# Исправление проблемы с датой

## Проблема

В БД дата сохранена как `2025-11-24`, а календарь на сайте показывает только ноябрь 2024. 

## Возможные причины

1. При добавлении через бота была выбрана дата 24.11.2025 (год впереди)
2. Календарь не показывает даты следующего года

## Решение

### Вариант 1: Исправить дату в БД (рекомендуется)

Если это опечатка, и должна быть дата 2024-11-24:

1. Откройте Drizzle Studio:
   ```bash
   npm run db:studio
   ```

2. Найдите запись в таблице `work_slots` с датой `2025-11-24`

3. Измените дату на `2024-11-24`

4. Или через SQL:
   ```sql
   UPDATE work_slots 
   SET work_date = '2024-11-24' 
   WHERE work_date = '2025-11-24';
   ```

### Вариант 2: Обновить календарь (если нужно поддерживать даты следующего года)

Если действительно нужна дата 2025 года, нужно обновить календарь, чтобы он показывал даты в любом году в пределах месяца от текущей даты.

## Быстрое решение

Выполните SQL запрос для исправления даты:

```sql
UPDATE work_slots 
SET work_date = '2024-11-24' 
WHERE work_date = '2025-11-24' AND master_id = 10;
```

Или через бота: удалите старую запись и добавьте новую с правильной датой (24.11.2024).
