## Компоненты, которые были реализованы

### 1. Header (components/Header.tsx)
- Логотип из `/logo/logo1.png` (скопирован в `public/logo/`)
- Размер логотипа: h-12 md:h-14
- Цвет шапки: #4A4A4A (серый)
- Sticky позиционирование

### 2. ServiceCategories (components/ServiceCategories.tsx)
- Отображение категорий услуг
- При клике на категорию показываются услуги этой категории
- Кнопка "Каталог" со стрелкой для возврата
- Интеграция с поиском

### 3. SearchBar (components/SearchBar.tsx)
- Поле поиска с белым фоном
- При фокусе открывается модальное окно
- Поиск по названию и описанию услуг

### 4. ServiceCard (components/ServiceCard.tsx)
- Карточки услуг с фотографиями
- Бежевый фон (#E8D5C4)
- Кнопка пастельного темно-зеленого цвета (#6B8E6B)
- Фотографии из папки `public/photo/`

### 5. Модальное окно поиска
- Полноэкранное модальное окно с белым фоном
- Кнопка возврата только со стрелкой (без текста)
- Поле поиска на всю ширину
- Результаты поиска отображаются ниже

## Цветовая палитра
- Фон сайта: белый (#FFFFFF)
- Шапка: #4A4A4A (серый)
- Карточки услуг: бежевый (#E8D5C4)
- Кнопки услуг: пастельный темно-зеленый (#6B8E6B)
- Текст на белом фоне: темно-серый (#2A2A2A, #4A4A4A)

## Шрифты
- Основной (sans): Inter
- С засечками (serif): Playfair Display
- Кнопки: font-body (Inter)
- Описания: font-serif (Playfair Display)

## Данные услуг
- Fallback данные в `app/page.tsx` (fallbackServices)
- Все услуги имеют imageUrl из папки `public/photo/`
- Категории: Стрижки, Окрашивание, Укладки, Маникюр, Педикюр, Макияж

## Функциональность
- Фильтрация услуг по категориям
- Поиск по услугам
- Модальное окно поиска при фокусе на поле
- Переключение между категориями и услугами
