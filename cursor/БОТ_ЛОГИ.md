# Где смотреть логи Telegram бота

## Способ 1: Терминал, где запущен бот

1. Откройте терминал (PowerShell или Command Prompt)
2. Перейдите в папку проекта:
   ```bash
   cd "C:\Users\Fortn\Downloads\Profit Club"
   ```
3. Запустите бота:
   ```bash
   npm run bot:start
   ```
4. В этом же терминале вы увидите все логи бота:
   - `Starting Telegram bot...`
   - `✅ Bot started successfully!`
   - `🚀 /start HANDLER CALLED!` (когда кто-то отправит /start)
   - `Generated verification code: ...`

## Способ 2: Проверить, запущен ли бот

Если бот уже запущен в фоне, вы можете:

1. Открыть новый терминал
2. Проверить процессы:
   ```powershell
   Get-Process | Where-Object {$_.ProcessName -eq "node"}
   ```

## Способ 3: Перезапустить бот с видимыми логами

1. Остановите все процессы Node.js:
   ```powershell
   Get-Process | Where-Object {$_.ProcessName -eq "node"} | Stop-Process -Force
   ```

2. Запустите бота в новом терминале (чтобы видеть логи):
   ```bash
   npm run bot:start
   ```

3. Оставьте этот терминал открытым - там будут все логи

## Что искать в логах:

- ✅ `Bot started successfully!` - бот запущен
- 🚀 `🚀 /start HANDLER CALLED!` - обработчик /start вызван
- 📝 `Generated verification code: 123456` - код сгенерирован
- ❌ Любые ошибки (Error, FATAL ERROR)

## Если бот не отвечает:

1. Проверьте, что в логах есть `Bot started successfully!`
2. Отправьте `/start` в Telegram
3. Проверьте, появилось ли `🚀 /start HANDLER CALLED!` в логах
4. Если нет - обработчик не вызывается (проблема с регистрацией)
5. Если есть, но код не отправляется - смотрите дальше в логах на ошибки
