## 🚀 Как начать тестирование

### 1️⃣ Тест Telegram-бота

**Откройте Telegram и найдите вашего бота**

1. Отправьте команду `/start`
   - Должно появиться главное меню (если ваш ID = 6140319824)
   - Если "нет доступа" - см. инструкцию ниже

2. Попробуйте добавить рабочий день:
   - Нажмите "➕ Добавить рабочее время"
   - Выберите мастера (например, "Маргарита Дереглазова")
   - Дата: `25.12.2024`
   - Время начала: `08:00`
   - Время окончания: `18:00`
   - Бот должен подтвердить создание

3. Посмотрите расписание:
   - Нажмите "📅 Просмотр расписания"
   - Должен отобразиться рабочий день

### 2️⃣ Тест API (быстрая проверка)

**Откройте в браузере:**

1. **Список мастеров:**

   ```
   http://localhost:3000/api/masters
   ```


### 3️⃣ Полный тест-сценарий

**Шаг 1: Добавьте рабочий день через бота**
- Мастер: Маргарита Дереглазова (ID: 10)
- Дата: 25.12.2024
- Время: 08:00 - 18:00

**Шаг 2: Проверьте доступные слоты**
Откройте в браузере:
```
http://localhost:3000/api/available-slots?masterId=10&serviceId=1&date=2024-12-25
```

Должны увидеть массив доступных слотов, например:
```json
[
  {"startTime": "08:00", "endTime": "09:00"},
  {"startTime": "08:30", "endTime": "09:30"},
  {"startTime": "09:00", "endTime": "10:00"},
  ...
]
```

**Шаг 3: Создайте запись (через curl или Postman)**
```bash
curl -X POST http://localhost:3000/api/appointments \
  -H "Content-Type: application/json" \
  -d '{
    "masterId": 10,
    "serviceId": 1,
    "appointmentDate": "2024-12-25",
    "startTime": "10:00",
    "clientName": "Тестовый клиент",
    "clientPhone": "+79001234567"
  }'
```

**Шаг 4: Проверьте, что слот занят**
Снова запросите доступные слоты - слот 10:00 должен исчезнуть!

**Шаг 5: Проверьте запись**
```
http://localhost:3000/api/appointments?masterId=10&date=2024-12-25
```

## ⚠️ Если что-то не работает

### Бот не отвечает
```bash
npm run bot:start
```

### API не работает
```bash
npm run dev
```

### Нет доступа к боту
Добавьте свой Telegram ID в таблицу `admins`:
```bash
npm run db:studio
```
Или через SQL:
```sql
INSERT INTO admins (telegram_id, full_name) 
VALUES ('ВАШ_TELEGRAM_ID', 'Ваше имя');
```
