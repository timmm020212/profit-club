# Настройка Drizzle Studio на local.drizzle.studio

## Шаг 1: Добавить запись в hosts файл

1. Откройте **Блокнот (Notepad) от имени администратора**:
   - Нажмите Win + X
   - Выберите "Windows Terminal (Admin)" или "Командная строка (администратор)"
   - Или найдите "Блокнот" в меню Пуск, правой кнопкой → "Запуск от имени администратора"

2. В Блокноте откройте файл:
   ```
   C:\Windows\System32\drivers\etc\hosts
   ```

3. В конец файла добавьте строку:
   ```
   127.0.0.1 local.drizzle.studio
   ```

4. Сохраните файл (Ctrl + S)

## Шаг 2: Запустить Drizzle Studio

В терминале проекта выполните:

```bash
npm run db:studio
```

## Шаг 3: Открыть в браузере

Откройте в браузере:
- **https://local.drizzle.studio**
- или **http://local.drizzle.studio:4983**

## Альтернатива: Использовать localhost

Если не хотите настраивать hosts файл, просто используйте:
- **http://localhost:4983**
