# 🚀 **Быстрый тест функциональности**

## 🧪 **Что проверить:**

### **1. Простой тест JavaScript:**
```
http://localhost:3000/test-simple
```
- Красная страница с синей кнопкой
- Должен показать alert при клике

### **2. Тест API:**
```
http://localhost:3000/api/test-simple
```
- Должен показать JSON данные
- Проверяет работу API

### **3. Основная админка:**
```
http://localhost:3000/admin
```
- Проверьте загрузку и стили

## 🔧 **Если не работает:**

### **Проблема 1: 404 на страницах**
```
Решение: Перезапустите сайт
.\start-site-fixed
```

### **Проблема 2: Alert не появляется**
```
Решение: Проблема с JavaScript
Проверьте консоль F12 на ошибки
```

### **Проблема 3: API не работает**
```
Решение: Проблема с сервером
Проверьте консоль сайта на ошибки
```

## 💡 **Что делать прямо сейчас:**

1. **Попробуйте** http://localhost:3000/test-simple
2. **Если 404** - перезапустите сайт
3. **Если работает** - JavaScript в порядке
4. **Проверьте** http://localhost:3000/api/test-simple
5. **Если все работает** - проблема в админ-компонентах

## 🎯 **Самый быстрый тест:**

```
1. Откройте http://localhost:3000/test-simple
2. Нажмите синюю кнопку
3. Должен появиться alert
```

Если alert появился - все работает! Если нет - проблема с JavaScript.

**Начните с простого теста!** 🧪
