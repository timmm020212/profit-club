# 🎨 **Отладка CSS/Tailwind**

## 🔍 **Проблема:** Сайт отображается текстом (без стилей)

## 🧪 **Тестирование:**

### **1. Откройте тестовую страницу:**
```
http://localhost:3000/test-page
```

Если на тестовой странице красный фон и белый текст - Tailwind работает.

### **2. Проверьте консоль браузера:**
- Нажмите F12
- Посмотрите вкладку Console и Network
- Ищите ошибки загрузки CSS файлов

### **3. Проверьте исходный код страницы:**
- Нажмите Ctrl+U
- Ищите подключение CSS файлов
- Проверьте что загружается tailwind

## 🔧 **Возможные решения:**

### **Если Tailwind не работает:**
```bash
# Пересобрать CSS
.\start-site-fixed
# В новом терминале:
npm-simple run build
```

### **Если проблема в админ-панели:**
Проверьте что админ-панель использует правильные классы:
```tsx
// Должно быть так:
<div className="min-h-screen bg-[#0B0B0C] text-white">
```

### **Если проблема в шрифтах:**
Проверьте что шрифты загружаются в globals.css

## 📋 **Что проверить:**

1. **Тестовая страница:** http://localhost:3000/test-page
2. **Консоль браузера:** F12 → Console
3. **Сеть:** F12 → Network (проверить загрузку CSS)
4. **Исходный код:** Ctrl+U

## 💡 **Если ничего не помогает:**
Возможно проблема в кэше браузера:
- Нажмите Ctrl+F5 для полной перезагрузки
- Или откройте в инкогнито режиме

**Проверьте сначала тестовую страницу!** 🎨
