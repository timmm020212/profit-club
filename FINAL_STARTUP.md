# 🚀 **Финальные команды для запуска**

## ✅ **Все исправлено и готово к запуску!**

## 🎯 **Правильные команды:**

### **1. Запуск базы данных**
```bash
net start postgresql-x64-15
```

### **2. Запуск сайта**
```bash
.\npm run dev
```

### **3. Запуск ботов**
```bash
.\npm run bot:masters
.\npm run bot:client
```

### **4. Или через bat файлы**
```bash
start-dev.bat
start-masters-bot.bat
start-client-bot.bat
```

## 🔄 **Полный порядок запуска:**

```bash
# Терминал 1: База данных
net start postgresql-x64-15

# Терминал 2: Сайт
.\npm run dev

# Терминал 3: Мастерский бот
.\npm run bot:masters

# Терминал 4: Клиентский бот
.\npm run bot:client
```

## 🌐 **Адреса после запуска:**
- **Сайт**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **Мастерский бот**: @ProfitClub_staff_bot
- **Клиентский бот**: @ProfitClub_club_bot

## 📁 **Готовые bat файлы:**
- `npm.bat` - обертка для npm
- `start-dev.bat` - запуск сайта
- `start-masters-bot.bat` - запуск мастерского бота
- `start-client-bot.bat` - запуск клиентского бота

## ✅ **Проверка работы:**
1. Запустите сайт и откройте http://localhost:3000/admin
2. Создайте рабочий день для мастера
3. Проверьте что мастер получает уведомление в Telegram
4. Подтвердите/отклоните рабочий день через бота
5. Проверьте что статус обновляется в админ-панели

**Все готово! Используйте `.\npm run dev` для запуска сайта!** 🎉
