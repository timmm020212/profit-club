# 🚀 **Команды через npx (работают в Windows)**

## ✅ **Используйте npx версии:**

### **Запуск сайта**
```
.\dev-npx
```

### **Запуск ботов**
```
.\masters-bot-npx
.\client-bot-npx
```

## 🔄 **Полный порядок запуска:**

```
# 1. База данных
net start postgresql-x64-15

# 2. Сайт
.\dev-npx

# 3. Мастерский бот
.\masters-bot-npx

# 4. Клиентский бот
.\client-bot-npx
```

## 🌐 **Адреса после запуска:**
- **Сайт**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **Мастерский бот**: @ProfitClub_staff_bot
- **Клиентский бот**: @ProfitClub_club_bot

## 📁 **Созданные файлы:**
- `dev-npx.bat` - запуск сайта через npx
- `masters-bot-npx.bat` - запуск мастерского бота через npx
- `client-bot-npx.bat` - запуск клиентского бота через npx

## 💡 **Почему npx работает:**
- npx кроссплатформенный
- Правильно обрабатывает пути в Windows
- Не зависит от Unix скриптов

**Попробуйте `.\dev-npx` - должно работать в Windows!** 🎉
