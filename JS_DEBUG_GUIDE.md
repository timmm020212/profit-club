# 🚨 **Проблема: JavaScript не работает**

## 🔍 **Диагностика:**

### **1. Проверьте новую страницу:**
```
http://localhost:3000/debug-js
```
- Две кнопки: inline onClick и addEventListener
- Если ни одна не работает - проблема глобальная

### **2. Проверьте консоль (F12):**
- Нажмите F12
- Посмотрите вкладку Console
- Ищите ошибки JavaScript
- Должно быть "Script loaded"

### **3. Проверьте настройки браузера:**
- JavaScript включен?
- Нет ли блокировщиков?
- Работает ли в другом браузере?

## 🔧 **Возможные причины:**

### **1. JavaScript отключен в браузере**
```
Решение: Включите JavaScript в настройках браузера
```

### **2. Проблема с React/Next.js**
```
Решение: Проверьте консоль на ошибки React
```

### **3. Проблема с Tailwind CSS**
```
Решение: Проблема может быть в CSS, а не JS
```

### **4. Проблема с сервером**
```
Решение: Перезапустите сайт полностью
```

## 💡 **Что проверить прямо сейчас:**

### **Шаг 1: Базовый тест**
```
1. Откройте http://localhost:3000/debug-js
2. Нажмите F12 → Console
3. Нажмите обе кнопки
4. Посмотрите есть ли сообщения в консоли
```

### **Шаг 2: Если ничего не работает**
```
1. Попробуйте другой браузер (Chrome, Firefox)
2. Проверьте включен ли JavaScript
3. Перезапустите сайт: .\start-site-fixed
```

### **Шаг 3: Если работает только одна кнопка**
```
- Inline onClick работает, а addEventListener нет → Проблема с DOM
- Обе не работают → Проблема с JavaScript глобально
```

## 🎯 **Самый важный тест:**

```
Откройте http://localhost:3000/debug-js
Нажмите F12
Нажмите кнопки
Посмотрите консоль
```

**Если даже inline onClick не работает - проблема с JavaScript в браузере или Next.js!** 🚨
