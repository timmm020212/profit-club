# 🚀 **Прямые команды (без npm)**

## ✅ **Минимальные команды без зависимостей**

## 🎯 **Используйте прямые bat файлы:**

### **Запуск сайта**
```
.\dev-direct
```

### **Запуск ботов**
```
.\masters-bot-direct
.\client-bot-direct
```

## 🔄 **Полный порядок запуска:**

```
# 1. База данных
net start postgresql-x64-15

# 2. Сайт
.\dev-direct

# 3. Мастерский бот
.\masters-bot-direct

# 4. Клиентский бот
.\client-bot-direct
```

## 🌐 **Адреса после запуска:**
- **Сайт**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **Мастерский бот**: @ProfitClub_staff_bot
- **Клиентский бот**: @ProfitClub_club_bot

## 📁 **Созданные файлы:**
- `dev-direct.bat` - прямой запуск сайта
- `masters-bot-direct.bat` - прямой запуск мастерского бота
- `client-bot-direct.bat` - прямой запуск клиентского бота

## 💡 **Почему это работает:**
- Не использует npm
- Прямой вызов node.exe
- Полные пути ко всем файлам

**Используйте `.\dev-direct` для запуска сайта без npm!** 🎉
