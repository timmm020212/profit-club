# ✅ **Исправлен простой знак рубля**

## 🎯 **Проблема решена:** Теперь используется обычный текстовый знак рубля как в тестовой услуге

## 🔧 **Что изменено:**

### **1. На сайте (`BookingServiceCard.tsx`):**
```typescript
{price && (
  <span className="text-lg font-semibold text-[#2A2A2A] whitespace-nowrap tabular-nums">
    {price}
  </span>
)}
```
- Убран дополнительный суффикс
- Цена отображается как есть из БД

### **2. В админке (`AdminSiteServicesManager.tsx`):**
```typescript
// Поле ввода - только цифры
<input
  value={draft.price}
  inputMode="numeric"
  onChange={(e) => {
    const next = e.target.value.replace(/\D+/g, "");
    setDraft((p) => ({ ...p, price: next }));
  }}
  placeholder="Напр: 1500"
/>

// Сохранение с пробелом и знаком
price: draft.price.replace(/\D+/g, "").trim() ? `${draft.price.replace(/\D+/g, "").trim()} ₽` : null,
```

### **3. Валидация обновлена:**
```typescript
const priceWithoutSuffix = next.price.replace(/[₽руб]/gi, "").trim();
```

## 💡 **Как теперь работает:**

### **В админке:**
1. Вводим "1500"
2. Сохраняем → в БД сохранится "1500 ₽"
3. На сайте отобразится "1500 ₽"

### **На сайте:**
- Отображается точно как в БД: "1500 ₽"
- Никаких дополнительных суффиксов

## 🧪 **Как проверить:**

### **1. Создайте услугу в админке:**
```
1. Откройте http://localhost:3000/admin/site
2. Создайте новую услугу
3. Введите цену: "1500"
4. Сохраните
```

### **2. Проверьте на сайте:**
```
1. Откройте http://localhost:3000/booking
2. Найдите новую услугу
3. Цена должна быть: "1500 ₽"
```

### **3. Сравните с тестовой:**
```
Тестовая: "2000 ₽"
Новая: "1500 ₽"
```

## ✅ **Результат:**

- ✅ **Простой текстовый знак:** "₽" как в тестовой услуге
- ✅ **С пробелом:** "1500 ₽" вместо "1500₽"
- ✅ **Единообразие:** Как в тестовых данных
- ✅ **Простота:** Никаких дополнительных стилей

**Теперь используется простой текстовый знак рубля как в тестовой услуге!** 🎉
