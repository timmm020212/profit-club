# 🚀 **РАБОЧИЕ КОМАНДЫ (ПРОВЕРЕНО)**

## ✅ **100% работающие команды:**

### **Запуск сайта**
```
.\start-site-fixed
```

### **Запуск ботов**
```
.\start-masters
.\start-client
```

## 🔄 **Полный порядок запуска:**

```
# 1. База данных
net start postgresql-x64-15

# 2. Сайт
.\start-site-fixed

# 3. Мастерский бот
.\start-masters

# 4. Клиентский бот
.\start-client
```
